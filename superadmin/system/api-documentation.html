Seller Integration API

REST API documentation — v1.0.0-dev · OAS 3.1

Base URL /api/v1
openapi.json
GET /health Health

Check API server health status and uptime.

Response 200

{
  "status": "healthy",
  "uptime": "14d 6h 32m",
  "version": "1.0.0-dev",
  "timestamp": "2026-02-26T09:30:00Z"
}
POST /api/v1/auth/login Login

Authenticate user and receive access token.

Request Body 

{
  "email": "admin@seller.id",
  "password": "your_password"
}

Response 200 

{
  "access_token": "eyJhbGciOiJIUzI1NiIs...",
  "token_type": "bearer",
  "expires_in": 3600,
  "user": { "id": 1, "email": "admin@seller.id", "role": "superadmin" }
}
POST /api/v1/auth/logout Logout

Invalidate current session access token.

Requires Bearer Token authentication

Response 200 

{ "message": "Successfully logged out" }
POST /api/v1/auth/logout-all Logout All

Invalidate all sessions and revoke all tokens for current user.

Requires Bearer Token authentication

Response 200 

{ "message": "All sessions revoked", "sessions_cleared": 3 }
GET /api/v1/auth/me Get Current User Info

Get information about the currently authenticated user.

Requires Bearer Token authentication

Response 200 

{
  "id": 1,
  "name": "Admin Master",
  "email": "admin@seller.id",
  "role": "superadmin",
  "created_at": "2025-01-15T08:00:00Z",
  "last_login": "2026-02-26T08:30:00Z"
}
POST /api/v1/auth/register Register User

Register a new user account (admin only).

Request Body 

{
  "name": "New Seller",
  "email": "seller@example.com",
  "password": "secure_password",
  "role": "seller"
}

Response 201 

{ "id": 42, "name": "New Seller", "email": "seller@example.com", "role": "seller" }
GET /api/v1/auth/users List Users

List all registered users with pagination.

Requires Bearer Token — superadmin only

Query Parameters

pageinteger, default: 1
limitinteger, default: 20
rolestring: superadmin | admin | seller

Response 200 

{
  "data": [
    { "id": 1, "name": "Admin Master", "role": "superadmin" },
    { "id": 2, "name": "Gadget Empire", "role": "seller" }
  ],
  "total": 892,
  "page": 1,
  "limit": 20
}
GET /api/v1/orders List Orders

Retrieve paginated list of orders with optional filters.

GET /api/v1/orders/{id} Get Order Detail

Get a single order by ID with full details including items and shipping.

POST /api/v1/orders/{id}/confirm Confirm Order

Confirm an order and move to packing status.

POST /api/v1/orders/sync Sync Orders from Marketplace

Trigger a manual sync of orders from all connected marketplaces.

GET /api/v1/shipping/track/{resi} Track Shipment

Track a shipment by receipt number.

POST /api/v1/shipping/labels Generate Shipping Labels

Generate PDF shipping labels for one or more orders.

POST /api/v1/shipping/bulk-track Bulk Track Shipments

Track multiple shipments in a single request (max 50).

GET /api/v1/webhooks List Webhooks

List all registered webhook endpoints.

POST /api/v1/webhooks Register Webhook

Register a new webhook endpoint to receive event notifications.

DELETE /api/v1/webhooks/{id} Delete Webhook

Delete a webhook endpoint by ID.
